# INSTRUCCIONES DE INSTALACIÓN

## Pasos para ejecutar la aplicación:

1. **Copiar el logo manualmente**:
   - Copie el archivo `logo_ESM Argentina.png` a la carpeta `src/assets/`
   - Este paso es necesario para que el logo aparezca correctamente en los componentes

2. **Instalar dependencias**:
   ```bash
   npm install
   ```

3. **Ejecutar la aplicación**:
   ```bash
   npm run dev
   ```

4. **Abrir en el navegador**:
   - La aplicación se abrirá automáticamente en `http://localhost:3000`
